# VPN Security Hub 🔒

เว็บไซต์ Responsive Design ที่อธิบายเกี่ยวกับ VPN ความเป็นส่วนตัว และความปลอดภัยบนอินเทอร์เน็ต

## ✨ ฟีเจอร์

- ✅ **Responsive Design** - ใช้ได้บนอุปกรณ์ทั้งหมด (มือถือ แท็บเล็ต เดสก์ทอป)
- ✅ **Modern UI** - ออกแบบสะอาดและมืออาชีพ
- ✅ **Smooth Animations** - แอนิเมชั่นที่ลื่นไหล
- ✅ **Mobile Menu** - เมนูแฮมเบอร์เกอร์สำหรับมือถือ
- ✅ **Contact Form** - แบบฟอร์มติดต่อพื้นฐาน
- ✅ **FAQ Section** - คำถามที่พบบ่อย

## 📁 โครงสร้างไฟล์

```
vpn-security-hub/
├── index.html       # ไฟล์ HTML หลัก
├── styles.css       # ไฟล์ CSS สำหรับสไตล์
├── script.js        # ไฟล์ JavaScript สำหรับ Interactivity
└── README.md        # ไฟล์นี้
```

## 🎨 Sections

1. **Navigation Bar** - เมนูนำทาง Sticky
2. **Hero Section** - ส่วนจุดเด่นหลัก
3. **Benefits** - ประโยชน์ของ VPN
4. **How to Use** - วิธีใช้ VPN 5 ขั้นตอน
5. **FAQ** - คำถามที่พบบ่อย
6. **Contact** - แบบฟอร์มติดต่อ
7. **Footer** - ส่วนท้าย

## 🚀 วิธีใช้งาน

### ท้องถิ่น (Local)

1. Clone repository:
```bash
git clone https://github.com/doheesyr61-alt/vpn-security-hub.git
cd vpn-security-hub
```

2. เปิดไฟล์ `index.html` ในเบราว์เซอร์

### GitHub Pages

1. ไปที่ Settings ของ Repository
2. เลือก "Pages" จากเมนูด้านซ้าย
3. เลือก Branch: `main`
4. เลือก Folder: `/ (root)`
5. บันทึก
6. เว็บไซต์จะขึ้นที่: `https://doheesyr61-alt.github.io/vpn-security-hub`

## 🎯 การปรับแต่ง

### เปลี่ยนสี
แก้ไขตัวแปร CSS ใน `styles.css`:

```css
:root {
    --primary-color: #0066cc;
    --secondary-color: #00ccff;
    --dark-bg: #0a0e27;
    --light-bg: #1a1f3a;
    --text-color: #ffffff;
    --accent-color: #ff6b6b;
}
```

### เปลี่ยนข้อมูล
แก้ไขข้อความ รูปภาพ และข้อมูลใน `index.html`

## 📱 Responsive Breakpoints

- **Desktop**: 1200px และมากกว่า
- **Tablet**: 768px - 1199px
- **Mobile**: 480px - 767px
- **Small Mobile**: ต่ำกว่า 480px

- CSS3 (Flexbox, Grid)
- Vanilla JavaScript
